Detailed Description

template<typename CHAR>
class z::file::reader< CHAR >

A class for iterative file loading.

The file::loader class loads as much of a file as possible before the specified timeout. If timeout is reached, closes the file. Then attempts to reopen file and continue loading at each subsequent call.

template<typename CHAR >
int z::file::reader< CHAR >::read ( const core::timeout &  time = core::timeout(-1))

Load the file contents.

Default loading function for wide character set.

Attempts to load as much of the file into memory as possible. If the timer runs out before that, returns 0 to indicate that it should be called again. Continues loading file contents on each subsequent call. If no parameter is given, then the entire file will be loaded into memory before returning.

Parameters
timean optional parameter to pause loading after a timeout.
Returns
-1 if unable to open the file at any point,
0 if open successful but not done loading, and
1 if done loading the file.
