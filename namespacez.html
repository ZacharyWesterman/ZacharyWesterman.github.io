typedef wchar_t z::Char

Default character type used by other libraries.

For some code, wide characters may not be required. By default, all related libraries use wchar_t characters. However, if the compiler flag Z_USE_NARROW is set, then char is used.

◆ Float

typedef float z::Float

Default floating-point type used by other libraries.

For some code, very high precision with numbers may be required. By default, all related libraries use single-precision floats. However, if the compiler flag Z_USE_DOUBLE is set, then double-precision floats are used instead.

◆ Int

typedef long z::Int

Default integer type used by other libraries.

For some code, very large integers may not be required. By default, all related libraries use long (usually 64-bit) integers. However, if the compiler flag Z_USE_SHORT_INT is set, then shorter (usu. 32-bit) integers are used. If he compiler flag Z_USE_LONG_INT is set, then longer (64 to 128 bit) integers are used.
