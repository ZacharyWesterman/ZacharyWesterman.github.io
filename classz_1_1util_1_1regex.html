Detailed Description

template<typename CHAR>
class z::util::regex< CHAR >

A template class for parsing and matching regular expressions.

Note that because the class requires seeking locations in a stream, it will not work nicely with the system::console class.

Search for a match to the pattern in the given input.

Attempt to match the pattern to characters in the given input stream. This method does not consume input and searches through the stream until a match is found. Because some streams may be very long, this method can optionally exit upon timeout.

Parameters
inputthe character stream to read from.
timethe amount of time to wait before returning.
Returns
1 if a match to the pattern was found. 0 if the search timed out before matching. -1 if the entire stream was searched and no match was found.
