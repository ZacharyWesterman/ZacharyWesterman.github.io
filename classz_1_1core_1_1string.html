Detailed Description

template<typename CHAR>
class z::core::string< CHAR >

A template class for character strings.

Strings are expected to be of either char or wchar_t types, but also works with other integer types.
Note that all integer types from wchar_t and larger are assumed to have UTF-16 encoding.

Additionally, this class is type-independent, so it handles all type conversions automatically.

Convert the string to its complex number equivalent.

This function splits a numeric string into its real and imaginary parts according to a+bi or a-bi, where the imaginary part is assumed to come after the real part. If no imaginary part is detected, then it is assumed to be 0. Also, note that if the base is greater than 17, then the complex part is assumed to be 0 as well, because I is a digit in that case.

◆ isValue()

template<typename CHAR >
bool z::core::string< CHAR >::isValue ( int  base = 10) const

Check if this string can be converted to a real number.

When converting a string to a number, alphabetic characters are assumed to represent numbers above 9. So the string "2EB0" in base 16 would convert to 11952. Valid numeric strings can have up to a single decimal point. A leading '-' is also allowed for negative numbers. Additionally, the character 'e' or 'E' is assumed to imply some exponent in the given base. This exponent can also be negative (e.g. 2.2e-3.1). It is important to note bases above 14 (such as hexadecimal) have E as a digit, so exponents are only interpreted for bases less than 15.

Parameters
basean optional parameter for choosing the base that the string is assumed to be in.
Returns
True if the string contains only characters valid for conversion into a value. False otherwise.
